# タスク管理ソフト試作
HTMLとCSSを使って作成した、タスク管理ソフトです

## 使い方
1. 入力欄にタスクを入力
2. 追加ボタンを押す
3．リストに追加される
4. リストのチェックボックスを押すとタスクの文字が暗くなる
5. リストのバツボタンを押すとタスクの削除ができる

## 想定した想定したユーザー操作
- 文字を入力して追加ボタンを押す
- 入力欄が空の状態で追加ボタンを押す
- 完了チェックボックスを入れずに削除する
- エラー表示中に再度追加ボタンを押す
- タスク追加後に該当タスクを即削除
- 完了状態のタスクのチェックボックスを再度押す
- 文字数オーバー中に文字を入力
